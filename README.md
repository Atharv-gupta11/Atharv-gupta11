`Python` `FastAPI` `LLaMA` `Groq API` `OpenCV` `Serper API`

### 🚦 Traffic Vehicle Tracking & Telemetry
Full tracking pipeline: YOLOv8 detects, ByteTrack assigns stable IDs through occlusion, Kalman filters smooth trajectories. Outputs **11,000+ row telemetry CSV** per video with per-frame speed and path length. Runs at **~19–20 FPS on 1080p**.

`YOLOv8` `ByteTrack` `OpenCV` `Python` `Kalman Filter` `Savitzky-Golay`

[![GitHub](https://img.shields.io/badge/View_Repo-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/Atharv-gupta11/Traffic-Vehicle-Tracking)
